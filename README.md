# Root$Wise Sprint 002 — Identity + Onboarding

This package keeps the approved hero image as the front door and adds working app navigation behind it.

## What is included
- Approved hero preserved
- Working top navigation hotspots
- Begin Your Journey flow
- Six identity paths
- Signup screen
- 10-question Financial Roots Assessment
- Root Score calculation
- Personalized dashboard shell
- Learn page
- Tools page
- Schools page
- My Journey profile page
- LocalStorage persistence
- Vercel-ready build

## Run locally
```powershell
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
npm install
npm run dev
```

## Deploy
Upload the contents of this folder to the existing RW GitHub repo and commit. Vercel will redeploy automatically.
